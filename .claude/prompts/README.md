## How to Use These Prompts

1. Browse the prompts in this directory
2. Open the relevant `.md` file
3. Copy the prompt template
4. Customize it with your specific details
5. Send to Claude in a new conversation

## Creating Your Own Prompts

You can add your own reusable prompts to this directory:

1. Create a new `.md` file in `.claude/prompts/`
2. Include:
   - Clear title and description
   - Usage instructions
   - The prompt template
   - Examples
3. Update this README with a link to your new prompt

## Tips for Good Prompts

- Be specific about what you want
- Provide relevant context
- Include examples when helpful
- Specify the output format you need
- State any constraints or requirements
